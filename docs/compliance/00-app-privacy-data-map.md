# App Privacy Data Map

## Dados coletados observados

- Conta: nome, email, telefone, data de nascimento
- Operação acadêmica: check-ins, frequência, progresso, graduações
- Conta vinculada: kids, teen, responsável
- Financeiro: status de assinatura/cobrança
- Telemetria: erros e métricas
- Push: tokens de dispositivo

## Finalidade

- Funcionalidade principal do app
- Segurança/autenticação
- Comunicação e notificações
- Cobrança/gestão contratual
- Observabilidade

## Compartilhamento provável

- Supabase
- Stripe
- Resend
- Sentry
- Vercel Analytics/Speed Insights

## Tracking

- Não há evidência suficiente para declarar tracking publicitário
