## 🚀 Comandos de Build

```bash
# 1. Setup Capacitor (primeira vez)
bash scripts/capacitor-setup.sh

# 2. Build web estático
CAPACITOR_BUILD=true pnpm build

# 3. Sync com plataformas nativas
npx cap sync

# 4. Abrir no Xcode (iOS)
npx cap open ios

# 5. Abrir no Android Studio
npx cap open android

# 6. Build iOS (via Xcode)
# Product → Archive → Distribute App → App Store Connect

# 7. Build Android AAB (via CLI)
cd android && ./gradlew bundleRelease
# Output: android/app/build/outputs/bundle/release/app-release.aab
```
