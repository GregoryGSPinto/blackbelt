# UX Audit

## Top fricções

- Falta de fluxo reviewer comprovado
- Exclusão pública antes estava quebrada
- Offline existe, mas experiência depende de backend para muitos caminhos
- Mobile build quebrado impede avaliação real do wrapper
- Claims de trial/billing ainda pedem simplificação

## Corrigido

- Exclusão pública funcional
- Links de termos corrigidos
- Mensagens de exclusão mais maduras

## Ainda depende de backend

- reviewer/demo tenant
- billing real
- validação operacional de sessão expirada em produção
